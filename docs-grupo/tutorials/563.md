# Download and install Godot

### Why this task matters
Godot is the free, open-source game engine you’ll use to open, explore, and edit your team’s StoryQuest.

Threadbare requires **Godot 4.5**. At the time of writing this is the latest version of Godot available.

### Steps

1. Visit the official Godot download page: go to <https://godotengine.org/download>.
3. Choose the correct download for your operating system. Do not click on the .NET version - it’s larger and not needed for your work.
4. Install Godot.
**Windows:** Extract the ZIP file to a folder like C:\Godot. Open the folder and run Godot.exe.
**macOS:** Open the .dmg file and drag Godot.app into your Applications folder.
**Linux:** Follow instructions based on your distro or simply extract and run the binary.
5. Launch Godot. Open the Godot app. You’ll see a Project Manager window..
6. (optional) Pin Godot for easy access. Right-click the Godot icon and pin it to your taskbar (Windows) or dock (macOS).

✅ Done! You’ve installed Godot and it’s ready to load your first project.
