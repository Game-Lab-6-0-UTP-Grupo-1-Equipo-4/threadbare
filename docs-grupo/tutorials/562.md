# Set up a GitHub account

### Why this task matters
All project files live on GitHub. Creating an account lets you accept your mentor or instructor’s invitation and eventually push your own changes to the shared game.
### Steps

1. Go to GitHub and sign up. Visit [https://github.com](https://github.com/). Click Sign Up in the top-right corner.
2. Pick a username your teammates will recognize. Use something clear and identifiable, like `jordan-dev` or `aya-gamequest`. Important: GitHub usernames are global and permanent, so pick something appropriate.
3. Enter an email address you check regularly. You’ll need this to receive your invite and future notifications. Set a strong password.
4. (optional) Skip setup questions. GitHub may ask things like "What do you plan to build?" You can skip these or answer briefly.
5. Verify your email. Open the email from GitHub and click the verification link. You should land on your GitHub dashboard.
6. Wait for an invitation to your team’s GitHub Organization. You will receive an invite to join a shared GitHub Org where your game project lives. **Accepting this invite is required before you can clone the repository in the next steps.** [Click here to find the invite in your GitHub profile](https://github.com/settings/organizations). 

✅ Done! You now have a GitHub account and are ready to join your team’s project.
