# Clone the forked repository to your computer

### Why this task matters
Cloning pulls the Threadbare fork from GitHub onto your machine, creating a local Git repo you can edit.
### Steps

1. **Create a folder for your StoryQuest.** Navigate to a location like **Documents** or **Desktop**. Right-click and select New > Folder. Name it something simple, like `storyquest` (no spaces or capital letters).
2. **Get the repository URL.** Open the repository on GitHub and look for the green dropdown box that says Code. Copy the HTTPS URL to your clipboard. It will look something like:
   ```
   https://github.com/OrgName/threadbare-yourquest.git
   ```

3. **Tell Git where to find the new folder.** There are two ways to do this:
   1. **Use the `cd` (change directory) command:** Open Git and type:
      ```
      cd folder-name
      ```
      to go into the new folder. For example:
      ```
      cd C:/Users/YourName/Desktop/StoryQuest
      ```
      Write this address down for future use. Tip: You can also type `dir` to list the contents of the current folder you are in.
   2. **Use the Git folder short-cut** 

-    (Windows-only): Right-click inside your new StoryQuest folder. Choose "Show more options". Select _Open Git Bash Here_. This opens a Git terminal directly in that folder. 
- (macOS): Control-click (or right-click) the folder and choose New Terminal at Folder (or Open in Terminal)

4. **Clone the repository into your folder.** Use this command:
   
   ```
   git clone https://github.com/OrgName/threadbare-yourquest.git
   ```

   Git will clone the repository as a new folder inside the one you made. You can just use the `cd` command once more to navigate into this folder.
5. **Check your status.** Run:
   ```
   git status
   ```
   You should see: `On branch main` and `nothing to commit`.

✅ Done! You now have a working copy of the project ready to open in Godot.
